  About Vision 360

Everything You Need to Know

Vision 360 is an immersive virtual campus explorer that lets anyone navigate the entire campus — from the main entrance to individual labs — without stepping foot on site.

How It Works

STEP 01

Choose a Location

Pick any scene from the home page — Main Campus, a department, classroom, or lab. Each card shows the scene type and how many hotspots it has.

STEP 02

Explore in 360°

Drag to look around in any direction. Use pinch-to-zoom on mobile or your scroll wheel on desktop for a fully immersive panoramic experience.

STEP 03

Navigate via Hotspots

Click glowing hotspot markers inside the viewer to instantly jump to linked scenes — moving from the main campus into departments, then into classrooms and labs.

Key Features

Full 360° Panorama
Powered by Pannellum for smooth, high-quality equirectangular panorama rendering.
Interactive Hotspots
Clickable navigation markers placed exactly where they belong using pitch & yaw coordinates.
Firebase Powered
All scene data is stored and served live from Firebase Firestore. No page reload needed to see updates.
Secure Admin Panel
Master admin can create, edit and delete scenes, place hotspots, and grant access to co-admins.
Mobile Responsive
Works seamlessly on phones, tablets, and desktops with gesture support for 360° dragging.
Instant Loading
Cinematic loading screen hides delays. Scenes appear only when fully ready for a polished experience.
